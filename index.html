Ajouter le depot dans F-Droid

  1. Ouvrez F-Droid.
  2. Allez dans Parametres puis Mes depots.
  3. Appuyez sur Ajouter un depot.
  4. Collez l URL de la variante voulue.
  5. Validez puis laissez F-Droid synchroniser l index.

Depot officiel F-Droid

Pour apparaitre dans le vrai depot F-Droid, il faut une soumission a fdroiddata et une build reproductible sans dependances Google/Firebase bloquantes. Tel quel, l application utilise encore Google Sign-In et Firebase, donc une variante dediee sans ces dependances sera probablement necessaire.

Avertissement third-party repository Le message affiche par F-Droid est normal pour un depot auto-heberge. Il indique que le depot est publie par TechniChauff et non audite directement par l equipe officielle F-Droid.
