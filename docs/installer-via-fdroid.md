# Installer via F-Droid

Cette page regroupe les deux dépôts F-Droid officiels de TechniChauff.

## Quel lien utiliser ?

- **Version complète** : pour lapplication standard
  - URL du dépôt : `https://pollomax847.github.io/TechniChauff/fdroid-complete/fdroid/repo`
- **Version technicien** : pour la variante allégée technicien
  - URL du dépôt : `https://pollomax847.github.io/TechniChauff/fdroid-technicien/fdroid/repo`

## Ajout dans F-Droid

1. Ouvrez lapplication F-Droid.
2. Allez dans **Paramètres** puis **Mes dépôts**.
3. Appuyez sur **Ajouter un dépôt**.
4. Collez lune des URLs ci-dessus.
5. Validez puis laissez F-Droid synchroniser lindex.

## Avertissement "third-party repository"

Lavertissement affiché par F-Droid est **normal**.

TechniChauff utilise ici un dépôt F-Droid auto-hébergé. Cela signifie que :

- le dépôt est publié par léquipe TechniChauff,
- il nest pas audité par léquipe officielle F-Droid,
- F-Droid affiche donc automatiquement un message de prudence.

Cet avertissement ne signifie pas que lapplication est malveillante. Il indique simplement que la distribution se fait via un dépôt tiers.
