# Installer via F-Droid

Cette page regroupe les deux depots F-Droid officiels de TechniChauff.

## Quel lien utiliser ?

- **Version complete** : pour l application standard
  - URL du depot : `https://pollomax847.github.io/TechniChauff/fdroid-complete/fdroid/repo`
- **Version technicien** : pour la variante allegee technicien
  - URL du depot : `https://pollomax847.github.io/TechniChauff/fdroid-technicien/fdroid/repo`

## Ajout dans F-Droid

1. Ouvrez l application F-Droid.
2. Allez dans **Parametres** puis **Mes depots**.
3. Appuyez sur **Ajouter un depot**.
4. Collez l une des URLs ci-dessus.
5. Validez puis laissez F-Droid synchroniser l index.

## Avertissement "third-party repository"

Cet avertissement affiche par F-Droid est normal.

TechniChauff utilise ici un depot F-Droid auto-heberge. Cela signifie que :

- le depot est publie par l equipe TechniChauff,
- il n est pas audite par l equipe officielle F-Droid,
- F-Droid affiche donc automatiquement un message de prudence.

Cet avertissement ne signifie pas que l application est malveillante. Il indique simplement que la distribution se fait via un depot tiers.
